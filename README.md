# E-Commerce Sales & Customer Analytics Dashboard (Power BI)

- Project Overview

This project demonstrates an end-to-end Business Intelligence workflow using Power BI. The objective was to analyze e-commerce sales data, identify business trends, and provide actionable insights through interactive dashboards.

- Project Workflow

-- Data Preparation

* Data cleaning using Power Query
* Null value handling
* Duplicate record validation
* Data type corrections
* Column formatting and optimization

-- Data Modeling

Implemented a Star Schema model consisting of:

* Fact_Sales
* Dim_Customers
* Dim_Products
* Dim_Date
* Marketing_Data

Relationships were designed using one-to-many cardinality with active single-direction filtering.

-- DAX Measures Created

Key metrics developed:

* Total Sales
* Total Profit
* Total Orders
* Total Quantity
* Profit Margin %
* Average Order Value (AOV)
* Total Customers
* Sales Growth %
